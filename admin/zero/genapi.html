# 生成api

由于自定义了goctl模板，所以在使用goctl生成api时，请指定goctl模板路径为：dev/goctl

例如（在项目根目录下执行）

goctl api go -api app/core/cmd/api/core.api -dir app/core/cmd/api -home ./build/dev/goctl
