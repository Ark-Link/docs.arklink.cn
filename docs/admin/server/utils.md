# 工具类

## 工具类分类

```
Utils:
    Auth.php                # JWT处理工具
    Captcha.php             # 验证码工具
    Client.php              # 获取客户端信息工具（可自行添加更多方法进去）
    DataFormat.php          # 数据格式化工具（驼峰转下划线、下划线转驼峰）
    Response.php            # 请求响应类
    SysLogger.php           # 日志类
    SysRedis.php            # Redis类
```
