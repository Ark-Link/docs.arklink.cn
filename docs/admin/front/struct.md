# 目录结构

``` bash
├─main.js (主要入口文件)
├─permission.js (路由守卫权限拦截)
├─App.vue (页面入口文件)
├─api (请求接口)
├─assets (资源目录)
├─components (通用组件)
├─config (全局配置)
├─core (核心文件)
│  ├─directives (vue指令)
│  ├─mixins  (vue mixin)
│  ├─permission (接口权限定义)
│  └─socket (WebSocket封装)
├─icons (svg图标，同vue-element-admin)
├─layout (布局文件)
├─router (路由定义)
│  └─modules (路由页面映射定义)
├─store (VueX)
├─styles (通用样式)
├─utils （工具类）
├─vendor （xlsx工具类）
└─views (功能基础页面)
